# 12-factor-app

See [presentation](https://maiermic.github.io/12-factor-app/) (work in progress).

## Development

```
nvm use
yarn install
yarn run start
```

### Create PDF
Local development server has to be running on port 1948

```
yarn run start
```

Then run:

```
yarn run create-pdf-local
```

## Deploy

```
yarn run deploy
```
